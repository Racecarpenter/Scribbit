# Scribb’it 🐸

A 1:1 creative game where users transform scribbles into art.

## Stack
- Next.js (App Router)
- Supabase (Auth, Postgres, Storage)
- Tailwind CSS

## Features
- Magic link auth
- Threads
- Scribble canvas
- Transform mode (underlay + color palette)
- Reveal animation

## Roadmap
- Reactions
- Push notifications
- Streak counter
- Group threads
